RSS Feed Configuration

Here you can configure the RSS feed for imLinks.
For more information visit the RSS Advisory Board website.

Note: The first time you visit this configuration form for the RSS Feed you have to click the Submit button to store the default values in the database.

 

RSS feed activated
Select Yes to turn RSS feed for imLinks on, select No to turn it off.

RSS feed title
The name of the channel. It's how people refer to your service.
If you have a website that contains the same information as your RSS file, the title of your channel should be the same as the title of your website.

RSS feed link
The URL to the website corresponding to the channel.

RSS feed description
Phrase or sentence describing the channel.

RSS feed image
Specifies a GIF, JPEG or PNG image that can be displayed with the channel.

RSS feed image width
Indicates the width of the image in pixels. Maximum value for width is 144.

RSS feed image height
Indicates the height of the image in pixels. Maximum value for height is 400.

RSS feed image title
Describes the image, it's used in the ALT attribute of the HTML <img> tag when the channel is rendered in HTML.

RSS feed image link
This is the URL of the site, when the channel is rendered, the image is a link to the site.
(Note, in practice the image <title> and <link> should have the same value as the channel's <title> and <link>).

RSS feed ttl
ttl stands for time to live. It's a number of minutes that indicates how long a channel can be cached before refreshing from the source.

RSS feed webmaster
Email address for person responsible for technical issues relating to channel.

RSS feed channel editor
Email address for person responsible for editorial content.

RSS feed category
Specify one or more categories that the channel belongs to.
Follows the same rules as the <item>-level category element.

RSS feed generator
String indicating the program used to generate the channel.

RSS feed copyright
Copyright notice for content in the channel.

RSS feed total links
Give the total number of links to display in RSS feed.

RSS feed offline message title
Enter the title for the message when RSS feed is deactived.

RSS feed offline message
Enter here an explanation for why the RSS feed has been deactivated.
