Atualizar para a versão 1.05 RC5

  1. Faça um backup do banco de dados do imLinks e um backup da pasta ../modules/wflinks no seu servidor.
  2. Desinstalar o WF-Links.
  3. Gravar em cima os arquivos com a nova versão em seu servidor.
  4. Instalar imLinks.
  5. Restaurar tabela do banco de dados a partir do ponto 1.
  6. Atualizar Módulo imLinks a partir da Administração.
  7. Nome do módulo foi alterado e a partir da versão 1.0.6 será conhecido como imLinks.

Ps: Esta tradução é uma cortesia da equipe de Suporte Brasileiro do ImpressCMS.
