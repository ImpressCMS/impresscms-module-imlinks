Instalação

Instruções válidas somente para uma nova instalação.
Se você quiser converter do WF-Links, selecione "Converter" no menu.

Lembre-se: É uma boa ideia fazer um backup de dados antes de instalar qualquer módulo.


Nova instalação do imLinks

  1. Carregar o módulo para o seu site

    Enviar para o 'modules/imlinks' e também o diretório 'uploads/images' dentro do seu {diretorio-raiz}.

    Se você quiser mudar o nome da pasta 'modules/imlinks' para por exemplo 'modules/weblinks' você tem que fazer isso antes de continuar com a instalação.

  2. Alterar e verificar as pastas e permissões

    Permitir gravação (CHMOD 777) nas seguintes pastas:

    {DiretórioRaiz}/uploads/images
    {DiretórioRaiz}/uploads/images/category
    {DiretórioRaiz}/uploads/images/category/thumbs
    {DiretórioRaiz}/uploads/images/flags
    {DiretórioRaiz}/uploads/images/flags/flags_small
    {DiretórioRaiz}/uploads/images/screenshots
    {DiretórioRaiz}/uploads/images/screenshots/thumbs
    {DiretórioRaiz}/uploads/images/thumbs
  3. Instalar o módulo

    Entre como administrador e vá até a página de Administração. Escolha Sistema --> módulos e instale o imLinks

  4. Configurar o módulo

    O Passo mais importante agora será a configuração das permissões para o grupo do módulo e seus blocos via Sistema --> grupos

    Lembre-se que você precisa criar pelo menos uma categoria para ajustar as permissões. Embora você possa definir os grupos de forma global para obter acesso ao módulo, lembre-se que as permissões de acesso para incluir links, moderar e aprovar irão depender destas definições extras que você poderá fazer apenas para alguns setores.

    Ps: Esta tradução é uma cortesia da equipe de Suporte Brasileiro do ImpressCMS.
