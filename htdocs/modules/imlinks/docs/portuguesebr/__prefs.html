Preferências

Primeiro devemos reservar um tempo para dar uma boa olhada sobre todas as configurações em Preferências do módulo e dentro do possível deverá executar já algumas modificações necessárias ou confirmar todos os conjuntos padrões. Neste local você irá aprender a configurar corretamente o módulo.

Popular:
Informe a quantidade de cliques que um link deverá ser considerado como Popular no seu site.
Padrão: 100 cliques

Links populares e novos:
Selecione como você irá mostrar os ícones populares e os novos links na sua listagem de links.
Temos as seguintes opções para você usar:

  • Mostrar como ícones
  • Mostrar como Texto
  • Não exibir

Padrão: Mostrar como ícones

Novos Links:
Informe o número de dias que um link será considerado como novo em seu site.
Padrão: 10 dias

Links atualizados:
Coloque o número de dias necessário para que um link seja considerado como Atualizado após a sua alteração.
Padrão: 10 dias

Quantidade de Links na lista:
Número de Links que será mostrado em cada categoria quando você estiver navegando.
Padrão: 10 links

Quantidade de Links no Admin:
Você deverá informar o número de novos links que serão mostrados na página principal do seu setor administrativo.
Padrão: 10 links

Ordem padrão dos links:
Precisa ser selecionado uma ordem para a exibição padrão dos links. Nós programamos 5 tipo de ordens para você utilizar de acordo com a sua preferência.

  • Título
  • Data de Envio
  • Avaliações
  • Popularidade
  • País

Para Cada opção poderá ser configurada como ascendente (A) ou descendente (D). Isto é muito importante para que você possa mudar o aspecto geral de tempos em tempos no seu site.
Padrão: Título (A)scendente.

Ordenar categorias por:
Escolha como as categorias e as subcategorias poderão ser ordenadas. Neste momento existem 2(dois) tipos possíveis.

  • Título
  • Importância

Padrão: Por Título

Sub-Categorias
Selecione SIM para mostrar sub-categorias. Ao Selecionar NÃO vai esconder as sub-categorias nas listagens.
Padrão: Não = Esconder sub-categorias.

Usar editor: (admin)
Você deverá escolher o editor de sua preferência na Administração.
Se você tem uma instalação 'simples' (e.g você está usando somente a classe de editores padrão do core, e não ativou os editores visuais fornecidos no pacote padrão), só poderá selecionar o DHTML Compacto para trabalhar corretamente.
Padrão: DHTML compacto.

Usar editor (usuário):
Você deverá escolher o editor de sua preferência para trabalhar no lado do Usuário.
Se você tem uma instalação 'simples' (e.g você está usando somente a classe de editores padrão do core, e não ativou os editores visuais fornecidos no pacote padrão), só poderá selecionar o DHTML Compacto para trabalhar corretamente. Você deverá tomar um pouco mais de cuidado com os grupos com permissão para usar editores visuais no lado do usuário. Como padrão, deixe habilitado apenas o editor DHTML para usuários. Apenas alguns grupos especiais deverão ter acesso ao editor visual. Isto é apenas uma sugestão, você poderá fazer da forma que achar melhor.
Padrão: DHTML compacto.

Usar captcha nos formulários:
Escolha Sim para usar o captcha nos forulários em que for enviar alguma informação no lado do usuário.
Padrão: Sim, ativar Captcha

Mostrar screenshots:
Selecione SIM caso você precise mostrar screenshots (pequenas imagens) para cada link no seu site
Padrão: Sim, mostrar pequenas imagens.

Usar Miniaturas:
Os seguintes tipos são suportados: JPG, GIF, PNG.
imLinks vai usar miniaturas para imagens. Se escolher Não irá usar a imagem orginal. Se o seu servidor não suportar miniaturas, use esta opção.
Padrão: Não usar miniaturas.

Atualizar Thumbnails:
Se escolher sim para atualizar o Thumbnail das imagens selecionadas, será atualizada em cada página que for visualizada, por outro lado, se escolher para não atualizar, sempre irá ver a primeira imagem em miniatura independentemente de quantas outras possam existir.
Padrão: Sim

Qualidade do Thumbnail:
Esta opção só surtirá efeito se estiver usando Miniaturas (Thumbnail) em seu site.

  • Baixa Qualidade: 0
  • Alta Qualidade: 100

Padrão: 100 ou claro, alta qualidade.

Manter Aspecto da Imagem:
Padrão: Não

Comprimento da Imagem:
Mostrar o comprimento do screenshot da imagem em (px).
Padrão: 100

Altura da Imagem:
Mostrar a altura do screenshot da Imagem em (px).
Padrão: 79

Upload em (KB):
Tamanho máximo permitido para os arquivos enviados.
Padrão: 200000

Comprimento da Imagem no Upload:
Comprimento máximo em (px) da imagem quando se envia o arquivo nos links.
Padrão: 600

Altura da Imagem no Upload:
Altura máxima em (px) quando se envia um arquivo de imagem ao servidor.
Padrão: 600

Usar Auto Screenshot:
Isto irá criar automaticamente uma tela (ScreenShot) com base na url. Este recurso de screenshots poderá ser ignorado e poderá não funcionar para todos os sites.
Padrão: No

Pasta Principal de Imagens:
Digite o URL sem uma barra final para informar em qual pasta irão ficar as suas imagens.
Padrão: modules/imlinks/images

Pasta de Upload de ScreenShots:
Digite o URL sem uma barra final.
Padrão: uploads/images/screenshots

Pasta de Upload de Imagem nas Categorias:
Digite o URL sem uma barra final.
Padrão: uploads/images/category

Diretório das bandeiras dos Paises
Digite o URL sem uma barra final.
Padrão: uploads/images/flags/flags_small

Relógio:
Relógio usado por padrão no ImLinks.
Aqui você pode configurar o modo como a data será formatada. Esta definição não é para os blocos do imLinks.
Veja também o Manual do PHP..
Padrão: D, d-M-Y

Administração de datas e horas:
O padrão na administração é Timestamp para o ImLinks.
Aqui você pode configurar o modo como a data é formatada para a administração imLinks.
Veja também o Manual do PHP.
Padrão: D, d-M-Y - G:i

Escolher o total de caracteres para a descrição:
Definir o total de caracteres para a descrição da categoria. Isto afeta e muito como você irá navegar nas categorias e talvez o lay-out do site. Você deve escolher um tamanho compatível com a sua necessidade.
Padrão: 200

Tamanho máximo de caracteres para meta palavras-chave:
Este é o tamanho máximo de caracteres que poderá ser utilizado para as meta palavras-chave.
Veja também Wikipedia para mais informações.
Padrão: 255

Mostrar outros links enviados pelo Autor:
Você deverá escolher Sim para que outros links do mesmo autor sejam também exibidos.
Padrão: Sim

Mostrar opção de visualização rápida:
Isto irá transformar a forma como podemos (ligar)on/off(desligar) a opção Rápida de Visualização.
Padrão: Não

Exibir Social Bookmarks:
Selecione Sim se quiser que um Social Bookmark seja exibido na página principal do link.
Padrão: Sim

Exibir Google PageRank™:

Padrão: Sim

Nota: Dependência O módulo Tag precisa estar instalado, caso contrário o formulário não irá apresentar este recurso.
Padrão: Não

Utilize o endereço e opções de mapas:
Selecione Sim para utilizar o endereço e o recurso de mapas. Dependendo da localização do link, isto poderá não funcionar adequadamente.
Padrão: Sim

Usar ICMS RSS:
Dependência - Escolhendo Sim você irá usar a classe de RSS do ImpressCMS. Caso escolha Não ovcê irá utilizar o módulo Brandycoke RSSfit (de qualquer maneira, este módulo precisa estar instalado e ativo para este recurso trabalhar corretamente).
Padrão: Sim

Total de Feeds:
Informe o valor padrão da página total de seus feeds.
Padrão: 15

Link para termos do glossário:
Dependência - O módulo imGlossary precisa estar instalado e ativo para este recurso funcionar corretamente.
Padrão: Não

Print page footer
Footer that will be printed for each link.
Padrão: <website_url>

Nome da pasta onde está o ImGlossary:
O nome da pasta onde está fisicamente o módulo imGlossary no seu site. Normalmente a pasta deverá ser imglossary mas se você instalou em outro local, fique muito atento aqui.
Padrão: imglossary

Logo print url
Url of the logo that will be printed at the top of the page.
Padrão: <website_url>/modules/wflinks/images/logo-en.gif
