Gestão de Links

 

Utilize o gerenciamento de link para ligar o formulário no momento em que você está criando um novo link.
Este formulário tem os seguintes campos/opções:

Título do Link
Digite aqui o título para o novo link.

Url do Link
Digite o url do novo link.
Clique no ícone do globo para abrir a url que você informou em seu navegador na guia/janela. Você pode usar esse botão ou ícone para verificar o url.

Link da categoria Principal
Selecione a categoria principal para o link.

Descrição do Link
Aqui você pode digitar uma descrição do novo link.

Palavra-chave
Neste campo você pode inserir as meta palavras-chave. Introduza as palavras-chave como: keyword1, keyword2, keyword3, ...
O total máximo de caracteres poderá ser definido em Preferências do módulo.

Link do screenshot da imagem
O Screenshot deve ser um link de imagem válido uploads/images/screenshots diretório (ex. shot.gif).
Escolher Não Mostrar imagem se você não quiser exibir uma tela.

Mapas do Google, Mapas Yahoo e o Multimap
Estes três campos podem ser usados para digitar um URL para um mapa on-line.
Clique no ícone apropriado atrás do campo para abrir uma nova aba/janela do navegador.
O url usado para esta ação é o url informado no campo.
Você pode usar estes ícones ou botões para controlar ou modificar a url do mapa que você informou.

Endereços de Campos
Use estes campos para adicionar Detalhes de Contato para o link. A forma como o endereço é formatado baseia-se no país selecionado.
Se você quiser um outro formato de endereço você deverá modificar o arquivo ../include/address.php
Para ver os tipos de formatos União Postal Universal website.
Endereços de e-mail podem ser inscritos de 2 maneiras:

  • nome@endereco.com
  • mailto:nome@endereco.com

Ambos os formatos serão automaticamente convertidos para: Nome [at] endereço [dot] com

Escolha a data de Publicação
Introduza a data e hora em que o link será publicado.

Data de expiração do Link
Use as opções aqui para definir uma data de expiração para o link. O link não será mais visível para os visitantes.
Esta opção poderá ser utilizada para remover a data de validade também.

Definir link offline
Escolha Sim para definir o link off-line. O link não será mais visível para os visitantes.
Padrão: Não

Definir o status como atualizado
Escolha Sim para definir como um link atualizado. Um ícone irá aparecer por trás do título do link para notificar o visitante que o conteúdo do link foi alterado.
Padrão: Não

Adicionar Discussão neste fórum
Selecionando um fórum em uso irá fazer surgir um ícone para criar uma discussão sobre este link.
