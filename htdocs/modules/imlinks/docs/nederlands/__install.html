Install


Deze instrukties zijn voor een nieuwe installatie.
Als u wilt converteren van WF-Links kies dan 'Converteren' uit het menu.

Opmerking: Het is altijd goed om eerst een database backup te maken alvorens een module te installeren.

Nieuwe installatie van imLinks

1. Uploaden van de module naar uw website

Upload de mappen 'modules/imlinks' en 'uploads/images' naar uw {icmss-rootdirectory} Indien u de map 'modules/imlinks' wilt veranderen in bijv. 'modules/weblinken', dan dient u dat eerst te doen, alvorens verder te gaan met de installatie.

2. Verander en controlleer de map rechten

CHMOD de volgende mappen 777:

{icms-rootdirectory}/uploads/images
{icms-rootdirectory}/uploads/images/category
{icms-rootdirectory}/uploads/images/category/thumbs
{icms-rootdirectory}/uploads/images/flags
{icms-rootdirectory}/uploads/images/flags/flags_small
{icms-rootdirectory}/uploads/images/screenshots
{icms-rootdirectory}/uploads/images/screenshots/thumbs
{icms-rootdirectory}/uploads/images/thumbs

3. Installeer de module

Login als administrator, ga naar de Administratie pagina, kies Modules en installeer imLinks.

4. Configureer de module

De meest belangrijke stap nu is om de groep- en blokrechten voor de module in te stellen via Systeem --> Groepen
