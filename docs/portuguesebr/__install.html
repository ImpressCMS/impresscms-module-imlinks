Essas instruções são para uma nova instalação.

Se você quiser converter de myLinks/Weblinks selecione 'Convert' no menu.

Lembrar: É uma boa ideia fazer um backup de dados antes de instalar qualquer módulo.

Fresh instalação do WF-Links

  1. Carregar o módulo para o seu site

    Upload o 'modules/wflinks' e 'uploads/images' diretório para o seu {raiz xoops-diretorio}.

    Se você quiser mudar o nome da pasta 'modules/wflinks' para o exemplo a seguir 'modules/weblinks' você tem que fazer isso antes de continuar com a instalação.

  2. Alterar e verificar pasta permissões

    CHMOD as seguintes pastas para 777:

    {xoops-rootdiretorio}/uploads/images
    {xoops-rootdiretorio}/uploads/images/category
    {xoops-rootdiretorio}/uploads/images/category/thumbs
    {xoops-rootdiretorio}/uploads/images/flags
    {xoops-rootdiretorio}/uploads/images/flags/flags_small
    {xoops-rootdiretorio}/uploads/images/screenshots
    {xoops-rootdiretorio}/uploads/images/screenshots/thumbs
    {xoops-rootdiretorio}/uploads/images/thumbs

  3. Instalar o módulo

    Login como administrador e digite XOOPS página Administração. Escolha Sistema --> modules e instalar WF-Links

  4. Configurar oe módulo

    O Passo mais importante agora será a configuração das permissões para o grupo do módulo e seus blocos via Sistema --> grupos
