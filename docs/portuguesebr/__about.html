imLinks

A equipe do WF-Projetos está feliz de entregar um outro módulo para a Comunidade ImpressCMS. Gostaríamos de 'lembrar' que este módulo esta não é uma cópia do PD-Links e foi iniciado a partir do Módulo WF-Downloads.

imLinks É um módulo que oferece mais funcionalidades do que o módulo original mylinks.

Você pode criar um vínculo com várias categorias e subcategorias em seus links.

Aqui está uma pequena selecção de recursos oferecidos:

  • Criar várias categorias e subcategorias para os seus links
  • Novas contribuições podem utilizar permissões por Categoria
  • Permissões para Moderação por Categoria
  • Ao Enviar links você poderá escolher se será validado antes ou se será aprovado automaticamente
  • Aprovação automática para alguns grupos selecionados
  • Validar links
  • Incluir ScreenShots (automático) para seus links
  • Incluir uma descrição em seus links
  • Incluir um endereço incl. Mapas do Google, Yahoo Maps e vCard
  • Opções de Impressão
  • Permitir ao usuário fazer classificações e comentários para seus links
  • Definir data de publicação e expiração para cada link (opcional)
  • E muito mais...


Ps: Esta tradução é uma cortesia da equipe de Suporte Brasileiro do ImpressCMS.
