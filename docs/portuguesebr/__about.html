imLinks

Estamos encantados de entregar um outro módulo para a comunidade do ImpressCMS. Este módulo é uma «fork» do WF-Links versão 1.06.

imLinks é um módulo criado para o sistema ImpressCMS que oferece mais funcionalidades do que o módulo original mylinks que fez parte das distribuições do Xoops mais antigas.

Você pode criar um link (vínculo) com várias seções de categorias e e subcategorias.

Aqui está uma pequena selecção de funcionalidades oferecidas:

  • Create multiple categories and subcategories for your links
  • Submission permissions per category
  • Moderation permissions per category
  • Choose to validate link submissions first or have them automaticaly accepted
  • Automatic approval for selected groups
  • Validate links
  • Add (automatic) screenshots to your links
  • Add a description to your links
  • Add an address incl. Google Maps,Yahoo Maps and vCard
  • Print option
  • Allow user ratings and comments for your links
  • Define publishing and expiration times for every link (optional)
  • Supports WYSIWYG editors: FCK Editor, Koivi, TinyMCE and TinyEditor
  • Links can be cloned
  • RSS feed of latest links
  • And many more . . .
