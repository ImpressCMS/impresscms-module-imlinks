imLinks

Estamos encantados de entregar um outro módulo para a comunidade do ImpressCMS. Este módulo é uma «fork» do WF-Links versão 1.06.

imLinks é um módulo criado para o sistema ImpressCMS que oferece mais funcionalidades do que o módulo original mylinks que fez parte das distribuições do Xoops mais antigas.

Você pode criar um link (vínculo) com várias seções de categorias e e subcategorias.

Aqui está uma pequena selecção de funcionalidades oferecidas:

  • Criar várias categorias e subcategorias para os seus links
  • Controle de permissões por categoria para novos envios de links
  • Controle sobre permissões para Moderação por categoria
  • Poderá Enviar links que necessitam de validação antes, ou com aprovação automática após o envio
  • Aprovação automática para alguns grupos selecionados
  • Validade e validação nos links
  • Incluir (automáticamente) screenshots para seus links
  • Adicionar uma descrição dos seus links
  • Adicionar um endereço especial tais como: Mapas do Google, Mapas Yahoo e vCard
  • Opções de Impressão
  • Permitir votações no lado do usuário e comentários para seus links
  • Definir data de publicação e expiração para cada link (opcional)
  • Suporte para vários editores WYSIWYG: FCK Editor, Koivi, TinyMCE e TinyEditor
  • Links podem ser clonados
  • RSS feed dos últimos links
  • E muitos mais . . .


Ps: Esta tradução é uma cortesia da equipe de Suporte Brasileiro do ImpressCMS.
