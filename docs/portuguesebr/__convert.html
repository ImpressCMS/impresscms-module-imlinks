Conversão do WF-Links 1.06 --> imLinks


Instruções para a conversão da versão 1.06 do WF-Links para o imLinks.
Se você quiser fazer uma nova instalação do imLinks selecione "Instalar" neste Guia.

Lembre-se: É uma boa ideia fazer um backup de dados antes de instalar qualquer módulo.

Nota: Ao fazer a conversão o script de atualização irá converter as tabelas do WF-Links no banco de dados em tabelas imLinks. Após a conversão não é possível usar WF-Links, porque estará faltando as tabelas. Se quiser manter o seu antigo módulo WF-Links trabalho, você precisa fazer um backup das tabelas do WF-Links antes de iniciar esta atualização e após finalizar este trabalho, restaurar as tabelas novamente em seu banco de dados. É possível ter imLinks e WF-Links funcionando ao mesmo tempo (mas não sabemos por que será que irá fazer isto). Lembre-se, se você optar por retornar o backup irá ter todos os links convertidos em duplicata.

Agora vamos ver abaixo um passo a passo sobre esta conversão e esperamos que você consiga executar todas as etapas de forma tranquila.


1. Criar um backup de dados e arquivos

Backup das tabelas do myLinks/Weblinks e também do seu banco de dados para garantir tudo caso algo de errado aconteça.

2. Carregar o módulo para o seu site via ftp ou qualquer outro métdo em seu servidor.

Carregar o diretório 'modules/imlinks' para o seu {DiretórioRaiz}.
Você não precisa fazer upload da pasta 'uploads/images' porque imLinks irá utilizar a mesma que era usado pelo módulo WF-Links. Se você quiser renomear a pasta do imLinks para outro nome desejado, você tem que fazer isso antes de iniciar a instalação do módulo. Você deve saber que poderá instalar o imLinks com o nome de pasta que desejar dentro do diretório modules.

3. Alterar e verificar a pasta e suas permissões

Executar o comando CHMOD 777 nas seguintes pastas:

{DiretórioRaiz}/uploads/images
{DiretórioRaiz}/uploads/images/category
{DiretórioRaiz}/uploads/images/category/thumbs
{DiretórioRaiz}/uploads/images/flags
{DiretórioRaiz}/uploads/images/flags/flags_small
{DiretórioRaiz}/uploads/images/screenshots
{DiretórioRaiz}/uploads/images/screenshots/thumbs
{DiretórioRaiz}/uploads/images/thumbs

Instalar o módulo

Faça o Login como administrador e vá até a página de Administração. Selecione Sistema -> módulos e instale o imLinks

Inicie o script de conversão

Aponte o seu browser para {DiretórioRaiz}/modules/imlinks/update.php e execute o script de atualização.
Siga as instruções fornecidas durante o processo de instalação.
Caso você tenha instalado o módulo em uma pasta diferente, troque o endereço do módulo para que o link de atualização funcione corretamente.
O script irá tentar determinar qual a versão ou versões do WF-links você tem.
Se o módulo estiver instalado, ele irá tentar a atualização.

  • Atualizar o módulo

    Retorne para o Sistema --> Módulos e execute a atualização do imLinks, caso contrário, os modelos(templates) ainda irão ficar com a versão anterior e as páginas não serão exibidas corretamente.

  • Configurar o módulo

    A coisa mais importante agora será a configuração das permissões para o grupo do módulo e seus blocos via Sistema --> Grupos

  • Restaurar ou remover myLinks/webLinks

    Se quiser continuar usando myLinks ou Weblinks além de imLinks, em seguida, restaure o seus dados do myLinks/Weblinks.
    A partir de agora você pode fazer backup na etapa 1. Se você não quiser usar esses, desativar o módulo velho e também deverá desinstalá-lo.

    • Ps: Esta tradução é uma cortesia da equipe de Suporte Brasileiro do ImpressCMS.
