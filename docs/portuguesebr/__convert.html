Instruções para a conversão do myLinks .

Se você quiser fazer uma nova instalação de imLinks selecione "Instalar".

Lembre-se: É também uma boa ideia fazer um backup de dados antes de instalar qualquer módulo.

Conversão do myLinks/webLinks ==> para imLinks


Nota: Ao fazer a conversão um script de atualização irá converter as tabelas do myLinks/Weblinks em tabelas do imLinks. Após a conversão não é possível usar o myLinks/Weblinks, porque estará faltando todas as tabelas. Se quiser manter o seu myLinks/Weblinks ainda trabalhando, você precisa fazer um backup das tabelas do myLinks/Weblinks antes de iniciar a atualização e após terminar a conversão para o imLinks, deverá restaurar as tabelas novamente. É possível ter imLinks e myLinks/Weblinks funcionando ao mesmo tempo (mas isto é uma coisa que só você deverá escolher).

Agora vamos ver abaixo um passo a passo sobre esta conversão e esperamos que você consiga executar todas as etapas de forma tranquila.

  1. Faça um backup

    Backup das tabelas do myLinks/Weblinks e também do seu banco de dados para garantir tudo caso algo de errado aconteça.

  2. Carregar o módulo para o seu site via ftp ou qualquer outro métdo em seu servidor.

    Enviar o novo módulo para dentro do diretório 'modules' e o diretório 'uploads' para o seu {DiretórioRaiz}

  3. Alterar e verificar permissões de pastas

    Executar o comando CHMOD 777 nas seguintes pastas:

    {DiretórioRaiz}/uploads/images
    {DiretórioRaiz}/uploads/images/category
    {DiretórioRaiz}/uploads/images/category/thumbs
    {DiretórioRaiz}/uploads/images/flags
    {DiretórioRaiz}/uploads/images/flags/flags_small
    {DiretórioRaiz}/uploads/images/screenshots
    {DiretórioRaiz}/uploads/images/screenshots/thumbs
    {DiretórioRaiz}/uploads/images/thumbs

  4. Instalar o módulo

    Faça o Login como administrador e vá até a página de Administração. Selecione Sistema -> módulos e instale o imLinks

  5. Inicie o script de conversão

    Aponte o seu browser para {DiretórioRaiz}/modules/imlinks/update.php e execute o script de atualização.
    Siga as instruções fornecidas durante o processo de instalação.
    O script irá tentar determinar qual a versão ou versões de myLinks ou Weblinks você tem.
    Se o módulo estiver instalado, ele irá tentar a atualização.

  6. Atualizar o módulo

    Retorne para o Sistema --> Módulos e execute a atualização do imLinks, caso contrário, os modelos(templates) ainda irão ficar com a versão anterior e as páginas não serão exibidas corretamente.

  7. Configurar o módulo

    A coisa mais importante agora será a configuração das permissões para o grupo do módulo e seus blocos via Sistema --> Grupos

  8. Restaurar ou remover myLinks/webLinks

    Se quiser continuar usando myLinks ou Weblinks além de imLinks, em seguida, restaure o seus dados do myLinks/Weblinks.
    A partir de agora você pode fazer backup na etapa 1. Se você não quiser usar esses, desativar o módulo velho e também deverá desinstalá-lo.
