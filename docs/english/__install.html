Install


These instructions are for a fresh install.
If you want to convert from WF-Links please select 'Convert' from the menu.

Remember: It is always a good idea to make a database backup before installing any modules.


Fresh installation of WF-Links

1. Upload the module to your website.

Upload the 'modules/imlinks' and 'uploads/images' folder to your {icms-rootdirectory}.

If you want to rename the folder 'modules/imlinks' in for example 'modules/links' you have to do so before you continue with the installation.

2. Change and verify folder permissions

CHMOD the following folders to 777:

{icms-rootdirectory}/uploads/images
{icms-rootdirectory}/uploads/images/category
{icms-rootdirectory}/uploads/images/category/thumbs
{icms-rootdirectory}/uploads/images/flags
{icms-rootdirectory}/uploads/images/flags/flags_small
{icms-rootdirectory}/uploads/images/screenshots
{icms-rootdirectory}/uploads/images/screenshots/thumbs
{icms-rootdirectory}/uploads/images/thumbs

3. Install the module

Login as administrator and enter the ImpressCMS Control Panel (administration).
Select Modules and install imLinks.

4. Configure the module

Most important step now will be to setup the group permissions for the module and its blocks via Blocks.
