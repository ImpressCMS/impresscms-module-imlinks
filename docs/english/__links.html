Link Management

Use the Link Management to call the submit form for creating a new link.

This form has the following fields/options:

Link title
Enter here the title for the new link.

Alternative title for url
Enter an alternative title for the link to be used in the url. When the option Use nice urls from Preferences is selected and this field is left empty, than the title of the link will be used.

Link url
Enter the url of the new link.
Click the globe icon to open a browser tab/window with the entered url. You can use this icon or button to check the url.

Link main category
Select the main category for the link.

Link description
Here you can enter a description of the new link.

Text options
With this option you can turn on/off linebreaks.
For the DHTML editor you have to turn this option on and probably for TinyMCE also.

Keywords
In this field you can enter the meta keywords. Enter the keywords as: keyword1, keyword2, keyword3, ...
The maximum amount of characters can be set in the Preferences.

Link screenshot image
Screenshot image must be a valid image link under uploads/images/screenshots directory (ex. shot.gif). Choose Show no image if you don't want to display a screenshot.

Google Maps, Yahoo Maps and Multimap
These three fields can be used to enter an url to an online map. Click the appropriate icon behind the field to open a new tab/window in the browser. The url used for this action is the url entered in the field. You can use these icons or buttons to check or modify the map-url entered.

Address fields
Use these fields for add Contact Details to the link. The way the address is formatted is based on the selected country. If you want another address format you have to modify the file ../include/address.php For address formats see the Universal Postal Union website.
Street 2 is an optional field. If field is left blank it will not be used.

Email addresses can be entered in 2 ways:

  • name@address.com
  • mailto:name@address.com

Both email formats are automatic converted to: name [at] address [dot] com

If no country is selected the address can not be formatted and will not be displayed.

Add to TomTom
Enter the latitude and longitude of the address entered above. You can find the geographic position of the address by using the website iTouchMap.com
Geographic latitude/longitude of the location should be in decimal degrees. May be negative; "N"/"S"/"W"/"E" suffix is not accepted.
Default geodetic datum is WGS-84.

The geographic position should be entered as follows:

  • Latitude: 57.2355591
  • Longitude: -2.3535921

The Add to TomTom option will only be visible when you have entered your TomTom API key under Preferences.

Set publish date
Enter the date and time when the link will be published.

Link expire date
Use the options here to set an expiration date for the link. The link will no longer be visible for visitors. This option can be used for removing the expiration date also.

Set link offline
Select Yes to set the link offline. The link will no longer be viewable for visitors.
Default: No

Set link status as updated
Select Yes to set the link as updated. An icon will appear behind the title of the link to notify visitors that the content of that link has been changed.
Default: No

Add Discuss in this Forum
Selecting the forum in use will result in an icon for discussing the link in that forum.

Submit New link as News item*
Select Yes to submit the new link as an article in the News module.
Default: No

Select News Category to submit News*
Select the News category for submitting the article.

News Title*
Give a title for the article or leave blank to use the link title.

 

* These fields appear when the News module is installed only.
